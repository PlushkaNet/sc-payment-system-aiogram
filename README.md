## Что это?
Телеграм-бот системы платежей, написанный с использованием aiogram 3, для локального с друзьями minecraft сервера SC.

## Запустить и попробовать
Для запуска достаточно создать .env файл с полем TOKEN и токеном telegram бота:
```
TOKEN=<YOUR-TELEGRAM-BOT-TOKEN>
```
Собрать проект можно командой:
```
$ pip install .
```
Либо, используя **uv**:
```
$ uv pip install .
```
И запустить командой:
```
$ bot
```

## Какие у него функции?
- переводы другим людям
- просмотр таблиц лидеров по валюте (SCC)
- создание уникальных товаров, которые можно оплатить более быстрым путём, чем при обычном переводе (работает криво)

Этот проект был разработан ради интереса и опыта, более не разрабатывается и не поддерживается.
